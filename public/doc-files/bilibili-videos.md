# 我的 B 站视频

在文档里直接嵌入 B 站视频播放器。只要把视频链接**单独放在一行**，就会自动渲染成播放器，支持 `BV` 链接和 `av` 链接。

## 直接贴链接

https://www.bilibili.com/video/BV1xx411c7mD

## 用链接语法加标题

也可以用 Markdown 链接语法，给视频加一个标题（会显示在播放器下方）：

[示例视频：点击播放](https://www.bilibili.com/video/BV1xx411c7mD)

## 指定分 P 和起始时间

链接里带 `?p=2` 或 `?t=30` 也会被识别：

[第二分 P](https://www.bilibili.com/video/BV1xx411c7mD?p=2)

> 把上面的链接替换成你自己的 B 站视频地址（格式：`bilibili.com/video/BV...`）即可。
